Object Detection

Our project implements real-time object detection using the YOLO (You Only Look Once) algorithm. It supports both image and video inputs, allowing users to detect objects efficiently. Leveraging YOLO's accuracy, the system offers quick and reliable detection results, making it suitable for various applications.
